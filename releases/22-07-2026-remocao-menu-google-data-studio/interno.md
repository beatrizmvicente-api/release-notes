## Em uma frase
O menu Google Data Studio saiu pra todas as contas agora que os dashboards viraram nativos no Portal.

## Antes → Depois
| Antes | Depois |
|-------|--------|
| Menu Google Data Studio aparecia pra todas as contas — item fixo, numerado | Sumiu pra todas as contas; só reaparece com a flag `google-data-studio` inserida manualmente pra um usuário |
| Dashboards viviam numa ferramenta externa (Google Data Studio), fora do Portal | Migrados nativamente pro menu **Dashboards** do Portal |

## Por que importa
O valor que o cliente enxerga está nos dados dele — e esses dados precisam morar **dentro da plataforma**, não numa ferramenta de fora. O Google Data Studio empurrava isso pra um ambiente externo, com experiência ruim, que furava a jornada logo no começo. O novo usuário era o mais penalizado: em vez de ver o valor da API4COM no próprio Portal, caía num lugar que não era o nosso e se perdia. Trazer os dashboards pra dentro do Portal fecha esse vazamento — o menu antigo ficou redundante e foi aposentado.

- **Não é bug.** O menu sumir é intencional, pra todas as contas.
- **A flag é só exceção.** Cliente que ainda depende do Google Data Studio antigo recebe a flag na mão, pra aquele usuário.
- **A numeração do menu mudou pra todo mundo** — os itens seguintes subiram uma posição, porque o item saiu da lista fixa.

## O que muda no dia a dia
- **Pro Suporte:** "sumiu o menu Google Data Studio" é esperado — os dashboards agora ficam no menu **Dashboards** do Portal. Só é caso de mexer se o cliente realmente precisar do antigo.
- **Pro CS:** use como sinal de evolução — os dados do cliente agora vivem no Portal, com jornada mais limpa pro novo usuário. Quem ainda depende do Google Data Studio antigo recebe a flag na mão: exceção, não padrão.
- **Pro Comercial:** vira argumento — o cliente vê o valor da API4COM dentro da própria plataforma, sem detour por ferramenta externa. Experiência mais redonda, principalmente pra quem está começando.
- **Pro cliente:** nada a comunicar; a experiência de dashboards continua, agora nativa no Portal.
