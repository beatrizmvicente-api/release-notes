## Em uma frase
O menu Google Data Studio saiu pra todas as contas agora que os dashboards viraram nativos no Portal.

## Antes → Depois
| Antes | Depois |
|-------|--------|
| Menu Google Data Studio aparecia pra todas as contas — item fixo, numerado | Sumiu pra todas as contas; só reaparece com a flag `google-data-studio` inserida manualmente pra um usuário |
| Dashboards viviam numa ferramenta externa (Google Data Studio), fora do Portal | Migrados nativamente pro menu **Dashboards** do Portal |

## Por que importa
Os dashboards dependiam de uma ferramenta externa ao Portal — um pé pra fora do produto. Agora que viraram nativos, o menu antigo do Google Data Studio foi aposentado: saiu pra todo mundo de uma vez. A flag ficou só como escape hatch — quando um cliente específico ainda precisa do Google Data Studio antigo, o time insere na mão pra aquele usuário.

- **Não é bug.** O menu sumir é intencional, pra todas as contas.
- **A numeração do menu mudou pra todo mundo** (os itens seguintes subiram uma posição), porque o item deixou de existir na lista fixa.

## O que muda no dia a dia
- **Pro Suporte:** "sumiu o menu Google Data Studio" é esperado — os dashboards agora ficam no menu **Dashboards** do Portal. Só é caso de mexer se o cliente realmente precisar do antigo.
- **Pro CS:** se algum cliente ainda depende do Google Data Studio antigo, o time insere a flag manual pra aquele usuário — é exceção, não o padrão.
- **Pro cliente:** nada a comunicar; a experiência de dashboards continua, agora nativa no Portal.
